# Minerals Revenue

***

### Overview

This document describes the user interface of the **Minerals Revenues** module within The Ghana Chamber of Mines web portal. The module enables authorized users (managers) to browse, search, and analyze mineral revenue data submitted by their respective mining companies. The data displayed in this module originates from the **Technical Statistics** section under Data Upload, where revenue statistics are first captured and submitted.

***

### 1. Data Source

The information presented in the Minerals Revenues module is sourced from the **Technical Statistics** upload section under Data Upload. Revenue statistics files uploaded through that module are processed and displayed here, providing a consolidated and analytical view of mineral production and export values across reporting periods.

### 2. Minerals Revenues List (View Page)

#### 2.1 Page Header

At the top of the content area, the page title **"Minerals Revenues List"** is displayed in bold, preceded by a back arrow icon (`←`) for navigation to the parent page. To the right of the title, the following controls are available:

* **Search Field**: A text input box with a light gray placeholder text "search", allowing users to filter the list of mineral revenue records by relevant keywords.
* **Refresh Button**: A circular refresh icon accompanied by the label "Refresh", enabling users to reload the list to reflect the latest data entries.
* **Filter by Company Dropdown**: A dropdown selection field labeled "Filter by Company" with a downward-facing chevron icon, allowing users to narrow the displayed records to a specific mining company.

#### 2.2 Data List

Below the header, a data grid displays mineral revenue records. The list includes the following column headers, each equipped with sort indicators (up/down arrows) for ascending and descending ordering:

* **Company Account**: Identifies the mining company associated with the revenue record.
* **Year**: The reporting year for the mineral revenue data.
* **Month**: The specific month during which the revenue data was recorded.
* **Quarter**: The fiscal quarter corresponding to the reporting period.
* **Mineral Production Value ($)**: The total monetary value of minerals produced during the reporting period, expressed in United States Dollars.
* **Mineral Exported Value ($)**: The total monetary value of minerals exported during the reporting period, expressed in United States Dollars.



When no mineral revenue records exist in the system, the data grid displays an empty state. Centered within the list area is a document inbox icon accompanied by the text **"No data"** in a light gray font. This indicates that no mineral revenue data has been submitted or recorded for the current view.
