# Sustainability

***

## PR Projects Module — User Interface Documentation

### Overview

This document describes the user interface of the **PR Projects** module within The Ghana Chamber of Mines web portal. The module enables authorized users (managers) to browse, search, and analyze public relations and community project data submitted by their respective mining companies. The data displayed in this module originates from the **Sustainability Reports** section under Data Upload, where project statistics are first captured and submitted.

***

### 1. PR Projects List (View Page)

<figure><img src="../.gitbook/assets/Screenshot 2026-07-20 192335.png" alt=""><figcaption></figcaption></figure>

#### 1.1 Page Header

At the top of the content area, the page title **"Pr Projects List"** is displayed in bold, preceded by a back arrow icon (`←`) for navigation to the parent page. To the right of the title, the following controls are available:

* **Search Field**: A text input box with a light gray placeholder text "search", allowing users to filter the list of PR project records by relevant keywords.
* **Refresh Button**: A circular refresh icon accompanied by the label "Refresh", enabling users to reload the list to reflect the latest data entries.

#### 1.2 Data List

Below the header, a data grid displays PR project records. The list includes the following column headers, each equipped with sort indicators (up/down arrows) for ascending and descending ordering:

* **Year**: The reporting year for the PR project data.
* **Month**: The specific month during which the project data was recorded.
* **Company Name**: The name of the mining company associated with the PR project record.
* **Project Name**: The specific name or title of the public relations or community project.
* **Project Location**: The geographical location where the project was implemented.
* **Project Type**: The classification or category of the project.
* **Project Start Date**: The commencement date of the project.
* **Project End Date**: The completion or anticipated completion date of the project.
* **Project Status**: The current state or progress of the project.
* **Total Amount Spent**: The total financial expenditure incurred on the project.
* **People Directly Impacted**: The number of individuals directly benefiting from or affected by the project.
* **People Indirectly Impacted**: The number of individuals indirectly benefiting from or affected by the project.

#### 1.3 Empty State

When no PR project records exist in the system, the data grid displays an empty state. Centered within the list area is a document inbox icon accompanied by the text **"No data"** in a light gray font. This indicates that no PR project data has been submitted or recorded for the current view.

***

### 2. Data Source

The information presented in the PR Projects module is sourced from the **Sustainability Reports** upload section under Data Upload. Project statistics files uploaded through that module are processed and displayed here, providing a consolidated and analytical view of mining company public relations and community projects across reporting periods.
