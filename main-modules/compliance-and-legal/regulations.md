# Regulations

***

## Regulations Module — User Interface Documentation

### Overview

This document describes the user interface of the **Regulations** module within The Ghana Chamber of Mines web portal. The module enables authorized users (managers) to browse, search, and track regulatory compliance data submitted by their respective mining companies. The module provides a comprehensive list view for monitoring all regulations and their compliance statuses.

***

### 1. Regulations List (View Page)

<figure><img src="../../.gitbook/assets/Screenshot 2026-07-20 205953.png" alt=""><figcaption><p>Regulations List</p></figcaption></figure>

#### 1.1 Page Header

At the top of the content area, the page title **"Regulations List"** is displayed in bold, preceded by a back arrow icon (`←`) for navigation to the parent page. To the right of the title, the following controls are available:

* **Search Field**: A text input box with a light gray placeholder text "search", allowing users to filter the list of regulations by relevant keywords.
* **Refresh Button**: A circular refresh icon accompanied by the label "Refresh", enabling users to reload the list to reflect the latest data entries.

#### 1.2 Data List

Below the header, a data grid displays regulation records. The list includes the following column headers, each equipped with sort indicators (up/down arrows) for ascending and descending ordering:

* **Regulation ID**: The unique identifier assigned to each regulation entry.
* **Title**: The title or name of the regulation.
* **Type**: The classification or category of the regulation.
* **Issuing Authority**: The governmental or regulatory body that issued the regulation.
* **Regulatory Body**: The specific regulatory organization responsible for overseeing the regulation.
* **Status**: The current compliance status of the regulation, displayed as a text label.
* **Compliance Deadline**: The date by which compliance with the regulation must be achieved, formatted as DD-MM-YYYY.
* **Compliance Status**: The current state of compliance with the regulation, displayed as a text label.
* **Compliance Officer**: The name of the individual responsible for ensuring compliance with the regulation.
* **Actions**: A three-dot menu icon (`⋯`) at the far right of each row, providing access to additional actions ( show- to preview)  for the respective regulation record.

#### 1.4 Pagination

At the bottom-right of the list area, pagination controls are present. The current view shows page "1" highlighted in a bordered box, with left and right arrow controls for navigating between pages.
