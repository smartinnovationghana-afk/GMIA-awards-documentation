# Adhoc

***

### Overview

This document describes the user interface of the **Adhoc Reports** module within The Ghana Chamber of Mines web portal. The module is designed to handle non-standard or special-purpose reports submitted by mining companies. Access to this module is contingent upon the prior upload of a designated template by the Chamber of Mines administrative team.

#### 1.1 Page Content

When a user navigates to the Adhoc Reports section without an available template, the system displays a centered informational message within the main content area. The page consists of the following elements:

* **Title**: The heading **"Adhoc Template Missing"** is displayed prominently in bold, centered text. This title clearly communicates the reason the module is currently inaccessible.
* **Explanatory Message Line 1**: Below the title, the text reads: _"A Template Has Not Been Uploaded Yet For The Adhoc Reports."_ This informs the user that the prerequisite template required for adhoc report submissions has not yet been configured in the system.
* **Explanatory Message Line 2**: The subsequent line states: _"Please Wait For Chamber Of Mines To Upload One Before Accessing This Section."_ This message directs the user to await administrative action from the Chamber of Mines before the module becomes available for use.



The template is only accessible when the Ghana Chamber of Mines uploads it. If the Chamber has not uploaded the template, you cannot submit ad hoc reports.
