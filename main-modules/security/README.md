# Security

## Security overview

The Security module provides a central place to review security-related reporting in The Ghana Chamber of Mines portal. It helps authorized users monitor submitted records, check reporting periods, and follow security incidents across member companies.

### What this module does

* Consolidates security reporting for review.
* Supports search and quick access to submitted records.
* Presents incident data by company, year, month, and quarter.

### How data flows

Security report data is captured through the incident reporting workflow and then displayed in the reporting area for review. This gives managers a single view of submitted security issues and their recorded occurrences.

### Available page

#### [Security Reports](security-reports.md)

Use **Security Reports** to browse submitted security records. The page shows key report fields such as company name, reporting period, security issue, and occurrence count.
