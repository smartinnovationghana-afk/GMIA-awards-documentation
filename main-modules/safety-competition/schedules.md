# Schedules

***

## Schedules Module — User Interface Documentation

### Overview

This document describes the user interface of the **Schedules** module within the Safety Competition section of The Ghana Chamber of Mines web portal. The module enables authorized users (managers) to browse, search, and view upcoming competition schedules in either tabular or calendar format. The module provides dual-view functionality, allowing users to toggle between a structured list view and a visual calendar representation of competition dates.

***

### 1. Upcoming Competitions — Table View

<figure><img src="../../.gitbook/assets/Screenshot 2026-07-20 212043.png" alt=""><figcaption><p>Tabular View of Upcoming Competitions</p></figcaption></figure>

#### 1.1 Page Header

At the top of the content area, the section title **"Upcoming Competitions"** is displayed in bold. A calendar icon is positioned on the far right of the header bar.

#### 1.2 View Toggle Controls

Below the header, two view options are presented as selectable tabs:

* **Table View**: The currently active view, indicated by a dark underline beneath the label. The tab includes a grid icon preceding the text "Table View".
* **Calendar View**: An alternative view option, indicated by a calendar icon preceding the text "Calendar View". Clicking this tab switches the display to a calendar-based visualization of competition schedules.

#### 1.3 Data List

Below the view toggle, a data grid displays upcoming competition records. The list includes the following column headers:

* **Title**: The name or title of the competition.
* **Competition Start Date**: The commencement date of the competition, formatted as DD-MM-YYYY.
* **Status**: The current state of the competition, displayed as a text badge.
* **Competition End Date**: The conclusion date of the competition, formatted as DD-MM-YYYY.

### 2. Upcoming Competitions — Calendar View

<figure><img src="../../.gitbook/assets/Screenshot 2026-07-20 212108.png" alt=""><figcaption><p>Calendar View for Upcoming Competitions</p></figcaption></figure>

#### 2.1 Page Header

The section title **"Upcoming Competitions"** remains consistent with the table view. The calendar icon on the far right of the header bar is retained.

#### 2.2 View Toggle Controls

The view toggle tabs are still present, with **"Calendar View"** now active (indicated by a dark underline) and **"Table View"** available as the alternative option.

#### 2.3 Calendar Controls

At the top-right of the calendar area, the following navigation controls are available:

* **Year Selector**: A dropdown field displaying "2026", allowing users to select a different year.
* **Month Selector**: A dropdown field displaying "Jul", allowing users to select a different month.
* **View Mode Buttons**: Two toggle buttons labeled "Month" and "Year", with "Month" currently active (indicated by a bordered outline), allowing users to switch between monthly and yearly calendar perspectives.

#### 2.4 Calendar Grid

The main content area displays a monthly calendar grid with the following characteristics:

* **Day Headers**: The days of the week are labeled across the top: Su (Sunday), Mo (Monday), Tu (Tuesday), We (Wednesday), Th (Thursday), Fr (Friday), Sa (Saturday).
* **Date Cells**: Individual days are displayed as numbered cells within the grid. Days from the previous month (28, 29, 30) and the following month (01) are shown in a lighter gray font to indicate they fall outside the current month.
* **Current Date Highlight**: The current date (20) is displayed in a dark red font within a shaded gray cell, indicating the present day.
* **Competition Event**: The competition "BEst" is displayed as a green highlighted bar spanning from July 8th to July 18th, visually representing the scheduled duration of the competition within the calendar.
